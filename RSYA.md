1. Открой без VPN и проверь главную, `/create/`, `/articles/`, `/faq/`, `/privacy/`, `/contacts/`, `/dlya-kogo/`
2. В Метрике есть визиты
3. В Вебмастере сайт подтверждён
4. В Timeweb включён редирект HTTP → HTTPS

## Как подать повторно

1. https://partner.yandex.ru → **Реклама на сайтах → Сайты**
2. `biznes-ip.ru` → **Действия → Отправить на модерацию**
3. Кнопка активна не раньше срока (для «иным причинам» — обычно **30 дней**)

## После одобрения (код уже готов)

1. В кабинете РСЯ создай 1–3 блока (например adaptive).
2. В GitHub Secrets / env сборки (или `.env.local` локально) задай:
   - `NEXT_PUBLIC_RSYA_ENABLED=true`
   - `NEXT_PUBLIC_RSYA_BLOCK_ARTICLES=R-A-XXXXXX-Y`
   - `NEXT_PUBLIC_RSYA_BLOCK_FAQ=R-A-XXXXXX-Y`
   - `NEXT_PUBLIC_RSYA_BLOCK_ABOUT=R-A-XXXXXX-Y`
3. `git push` / пересобери — блоки появятся внизу статей, FAQ и «О проекте».
4. **Не** ставить рекламу на `/create` (форма PDF).
5. Первую неделю смотри, не мешает ли вёрстке.
